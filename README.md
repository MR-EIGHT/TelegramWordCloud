# TelegramWordCloud

Simple Word Cloud Generator for Telegram chats.



# Supported Languages
- English
- Farsi

# Usage
1- Export any Telegram chat that you prefer as json file.  
2- Give WordCloud.py the exported chat's address.  
3- Wait for the WorldCloud to generate the picture.
# Requirements
- hazm
- matplotlib
- WordCloudFa
- numpy
- nltk
